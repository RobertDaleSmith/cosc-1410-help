Computer Science I (C++)
Supplemental Resources

I have talk to several people that are having a difficult time in this course, so I decided that I would provide some supplemental resources for everyone. What has helped me to learn in my prior programming courses were good example programs that I could just tweak and modify to see how it all works.


C++ Source Code

I will be posting all of this semesters assignments two weeks after they are due. These assignments have been programmed to exact assignment specs. Download the *.cpp so that you can view the code, edit it, and compile it. Use these to figure out assignments you didn't fully understand. This will help you prepare for the exam as well.

Along with the assignments I will post some examples that the TA went over in the lab. Since I can not post the latest assignments, I have dug up some projects that I did in a C++ course in 2007.

Note: To download files, just right click on the file link, and then click Save As.
